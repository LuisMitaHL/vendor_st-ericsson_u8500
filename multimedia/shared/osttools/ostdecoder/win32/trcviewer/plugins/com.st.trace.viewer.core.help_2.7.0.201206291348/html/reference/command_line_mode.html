Launching Trace Viewer from command-line

Trace Viewer can be launched from the command-line using the same options as decoder, but preceded by "--decoder" (an example is given below in the "Tips" section). Specific options can also be provided (list of available options is in the section below).

Once the viewer has been launched, the arguments are stored in order to be reused from the GUI. So if you click on the "Open Trace Stream..." button, the dialog box will be pre-filled with the last parameters you used (either command-line or GUI).



Command-line options

DO NOT FORGET to PRECEDE your list of options with "--decoder". This activates the command-line mode in Trace Viewer.

Below is the current list of options specific to Trace Viewer in command-line mode:

  • --pause

    Activates the "Pause on startup" option in Trace Viewer. The viewer will start the parsing in pause mode. For more information please refer to the Tips section of the Getting Started documentation page.

The current list of options available for the command-line decoder can be obtained calling it with the "--full-help" option.



Tips

  1. Help:

    Use "--full-help" or "./trcviewer.[sh|bat] --decoder --full-help" to print the help. Note that the "--decoder" option help is not detailed there.

  2. Example:

    "./trcviewer.[sh|bat] --decoder -h -a -r -D mydictionaries.txt mytrace.bin"
