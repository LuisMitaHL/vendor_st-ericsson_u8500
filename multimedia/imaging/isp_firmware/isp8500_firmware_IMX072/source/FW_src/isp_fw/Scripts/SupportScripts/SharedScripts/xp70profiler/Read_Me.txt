Procedure to generate the result file:

1. Get the trace file (.txt)

2. Run the python script i.e xp70_profiler.py as given below
   In the command prompt,
   <complete_path_and_name_of_python_script> <complete_path_and_name_of_trace_file> <complete_path_of_result_file>
   For eg.
   C:\xp70_profiler\..\..\xp70_profiler.py C:\xp70_profiler\..\..\..\trace.txt C:\xp70_profiler\..\..\..\results

   The result file would be generated with the name "result.xls" in the folder "C:\xp70_profiler\..\..\..\results".

Note: The folder "C:\xp70_profiler\..\..\..\results" would be created, if not present.
      Import the python modules, if not present
