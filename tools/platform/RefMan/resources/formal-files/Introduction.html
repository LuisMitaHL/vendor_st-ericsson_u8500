Introduction

About this Document

Purpose

This document is a reference manual for the ST-Ericsson Linux Platform used for developing applications with ST-Ericsson's mobile technology platforms. The purpose of this document is to supply comprehensive reference information and use cases to speed up the development work.

Target Audience

The target audience for this document is developers of native-code applications with ST-Ericsson's mobile technology platforms.

Structure and Contents

The functionality is logically divided into a number of categories. Each category manages one or many related groups of functions.

Code Samples

Code samples are delivered as part of the ST-Ericsson SDK and shall be used as examples. The code shall be adopted to the actual environment when used, for instance coding guidelines and undesired event handling.
